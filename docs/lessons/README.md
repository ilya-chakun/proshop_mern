# docs/lessons

This folder stores short investigation notes, debugging logs, and lessons
learned during AI-agent sessions or manual debugging.

## Rules

- **No secrets.** Never store credentials, tokens, or private keys here.
- Keep entries short and timestamped (use filenames like `YYYY-MM-DD-topic.md`).
- The goal is to help future sessions (human or AI) understand previous
  decisions quickly without re-investigating from scratch.
